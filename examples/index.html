Persian React Date-picker

persian-react-datepicker is React.js component that helps you to create datepickers based on Persian calendar

Installation

You can install this package using npm:

npm install --save react-persian-datepicker

It would be easy to destruct Calendar component from the package:

import { Calendar } from 'react-persian-datepicker';

Basic Calendar

You can create a persian date picker by using the Calendar component. The required props are as following:

  1. onSelect: Invoked when a date is selected and the moment object will be passed.
  2. prevMonth: The element for navigating to the previous month button.
  3. nextMonth: The element for navigating to the next month button.
const el = document.getElementById('basic-calendar');
const result = document.getElementById('basic-calendar-result');
const prev = <i className="fa fa-arrow-left"></i>;
const next = <i className="fa fa-arrow-right"></i>;
const onSelect = date => {
    result.textContent = date.format('jD/jMM/jYYYY');
};

React.render(<Calendar
    onSelect={onSelect}
    prevMonthElement={prev}
    nextMonthElement={next}
    />, el);

Date Picker Input

You can create a persian date picker by using the DatePicker component. Actually, it creates two inputs, properties of which can be customized through resultInput and formatterInput. resultInput is created in order to store the selected output in proper format. (This makes it easy to integrate with redux-form). formatterInput is the input that you see below. Either typing and selecting date on calendar can change it and the resultInput will be updated. Use your browser's element inspector to see the hidden input beside the formatter input. The DatePicker's props are as following:

  1. inputFormat (default: 'jYYYY/jMM/jDD'): A moment-jalaali string format. It should have to be a user-friendly format. (Jalali Recommended)
  2. outputFormat (default: 'YYYY-MM-DD'): A moment-jalaali string format. It should have to be a server-friendly format. (Gregorian Recommended)
  3. resultInput: The props of the result input
  4. formatterInput: The props of formatter input
const el = document.getElementById('datepicker');

ReactDOM.render(<DatePicker
    resultInput={{ type: 'hidden' }} />, el);

Default Month

You can set the default shown month on the date picker by setting a moment object to month prop:

const el = document.getElementById('default-month-calendar');

React.render(<Calendar
    month={moment('1372/06', 'jYYYY/jMM')}
    />, el);

Minimum and Maximum selectable dates

You can restrict selectable dates by using following props:

  1. min: The minimum selectable date. It should be a momentobject.
  2. max: The maximum selectable date. It should be a moment object.
const el = document.getElementById('basic-calendar');

React.render(<Calendar
    min={moment()} /* Now (Today) */
    max={moment().add(10, 'days')} /* 10 Days later */
    />, el);
